# Tabero 개인정보 처리방침

  **시행일: 2025년 6월 22일**

  Tabero(이하 "회사")는 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 개인정보 처리방침은 Apple의 개인정보 가이드라인에 따라
  작성되었습니다.

  ## 1. 수집하는 데이터 유형

  ### 1.1 식별자
  - **사용자 ID**: UUID (Universally Unique Identifier)
    - 목적: 사용자별 서비스 제공 및 앱 기능 사용
    - 사용자와 연결되지 않음

  ### 1.2 사용 데이터
  - **제품 상호작용**: 앱 내 탭, 기능 사용 빈도
    - 목적: 앱 성능 개선 및 사용자 경험 향상
    - 사용자와 연결되지 않음

  ### 1.3 진단
  - **충돌 데이터**: 앱 충돌 시 발생하는 기술적 정보
  - **성능 데이터**: 앱 로딩 시간, 번역 처리 시간
    - 목적: 앱 안정성 개선
    - 사용자와 연결되지 않음

  ### 1.4 기타 데이터
  - **앱 설정 정보**: 선택한 국적, 여행 국가, 언어 설정
  - **요청 횟수**: 번역 서비스 사용 횟수 (request_count)
    - 목적: 서비스 제한 관리 및 사용 통계
    - 사용자와 연결됨 (UUID 기반)

  ## 2. 수집하지 않는 데이터

  ### 명시적으로 수집하지 않는 정보:
  - **연락처 정보**: 이름, 이메일, 전화번호
  - **건강 및 피트니스**: 건강 관련 데이터
  - **재무 정보**: 결제 정보, 신용카드 정보
  - **위치**: 정확한 위치, 대략적인 위치
  - **민감한 정보**: 인종, 종교, 성적 지향 등
  - **연락처**: 주소록, 연락처 목록
  - **사용자 콘텐츠**: 사진, 비디오 (번역 후 즉시 삭제)

  ## 3. 데이터 사용 방법

  ### 3.1 앱 기능
  - 메뉴 번역 서비스 제공
  - 사용자 설정 저장 및 적용
  - 번역 이력 관리 (로컬 저장)

  ### 3.2 분석
  - 익명화된 사용 통계 분석
  - 서비스 개선을 위한 데이터 분석
  - 모든 분석 데이터는 사용자와 연결되지 않음

  ### 3.3 제3자 광고
  - Google AdMob을 통한 광고 표시
  - 광고 식별자(IDFA) 사용
  - 개인화된 광고 표시 가능
  - 사용자는 기기 설정에서 광고 추적 제한 가능

  ## 4. 데이터 저장 및 보안

  ### 4.1 데이터 저장 위치
  - **서버**: Supabase 클라우드 (UUID, 사용 횟수만 저장)
  - **로컬**: 사용자 기기 (설정, 번역 이력)
  - **임시 처리**: 메뉴 사진 (번역 중에만, 즉시 삭제)

  ### 4.2 보안 조치
  - 모든 서버 통신은 HTTPS 암호화
  - 로컬 데이터는 iOS 샌드박스 내 안전하게 저장
  - 최소한의 데이터만 수집

  ## 5. 사용자 권리

  ### 5.1 데이터 접근 및 수정
  - 앱 내 설정에서 모든 개인 데이터 확인 가능
  - 언제든지 설정 변경 가능

  ### 5.2 데이터 삭제
  - 앱 삭제 시 모든 로컬 데이터 자동 삭제
  - 서버 데이터 삭제 요청: indiveloper@gmail.com

  ### 5.3 데이터 이동성
  - 번역 이력 내보내기 기능 제공 (향후 업데이트)

  ## 6. 제3자 서비스

  ### 6.1 Google Gemini AI
  - 용도: 메뉴 이미지 번역
  - 공유 데이터: 메뉴 사진 (임시)
  - 개인정보 미포함

  ### 6.2 Google AdMob
  - 용도: 광고 표시
  - 공유 데이터: 광고 식별자
  - AdMob 개인정보 처리방침 적용

  ### 6.3 Firebase
  - 용도: 앱 성능 모니터링, 충돌 보고서
  - 공유 데이터: 기기 정보, 충돌 로그
  - Firebase 개인정보 처리방침 적용

  ## 7. 데이터 보유 기간

  - **UUID 및 사용 정보**: 마지막 사용 후 1년
  - **로컬 데이터**: 앱 삭제 시까지
  - **임시 데이터**: 처리 완료 즉시 삭제

  ## 8. 아동 개인정보 보호

  - 중요 변경사항은 앱 실행 시 동의 요청
